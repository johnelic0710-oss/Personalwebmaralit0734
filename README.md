John Elic Personal Website
This project is a personal website presented in a slide-based format.
It contains my basic information, family background, hobbies, and contact details.
The website is designed to follow the given guidelines for a clean and organized student profile presentation.
Contents
Slide 1 – Student Profile
Slide 2 – Detailed Information
Slide 3 – Family
Slide 4 – Achievements
Slide 5 – Hobbies
Slide 6 – Contact Details
Features
Simple and clean layout
Slide navigation (Next & Previous buttons)
Dot navigation at the bottom
Smooth slide transition effect
Responsive design (works on phone and laptop)
How to View
Open the project folder.
Click and open the file index.html in any web browser.
